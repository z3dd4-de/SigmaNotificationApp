Hilfe

1. Voraussetzungen

  • Sigma Docking Station TL2012

  • Sigma Sport Tacho xx.12*

  • Windows 10/11 64bit

  • Empfohlen: BikeDB 2025

* Aktuell werden BC12.12, BC16.12 und BC16.12 STS unterstützt

Es sollten alle Tachos der Topline 2009/2012-Reihe unterstützt werden. Wer einen anderen davon zum Laufen bekommt (sieht Github-Link in den Quellen), kann mir gerne die ID mitteilen, damit ich den allgemein verfügbar machen kann (siehe Bekannte Tacho-IDs).



2. Installation

ZIP-Installationspaket von Github herunterladen, an beliebigen Ort entpacken.

„SigmaNotificationApp.exe“ starten, nach Bedarf Link anlegen.



3. Verwendung

Die Anwendung starten wenn die Sigma Docking Station angeschlossen ist. Tacho mit Fahrdaten einlegen. Sobald die Meldung erscheint, dass ein Tacho erkannt wurde, können die Daten importiert werden.




Entweder im Menü Tacho → Auslesen drücken oder auf das Fahrrad-Symbol klicken.

Jetzt sollten alle Felder mit den verfügbaren Daten befüllt worden sein (abhängig vom verwendeten Tachotyp).

Diese Daten können durch Klicken des „Speichern“-Button gesichert werden.



Die eigentliche Aufgabe dieser Anwendung ist nur das Auslesen der Tachodaten. Um sie sinnvoll verwenden zu können, wird empfohlen, dass die BikeDB 2025 zur Verwaltung der Fahrraddaten benutzt wird. Dies ist eine mächtige Verwaltungssoftware für alles mögliche: verschiedene Fahrräder, Strecken, Fahrten, etc.



4. Speicher-Format

In der Version 1.0 erzeugt die Anwendung nur eine JSON-Datei „bikedata.json“, die jedes Mal überschrieben wird. Die eigentliche Aufgabe dieser Anwendung ist nämlich die Verwendung innerhalb der BikeDB 2025, die ebenfalls auf meinem Github-Account kostenlos verfügbar ist (siehe Quellen).



Wer eine eigene Lösung programmieren will, findet hier die C#-Klasse, die zum Serialisieren der JSON-Datei verwendet wird:

public class RideData

{

public string TachoName { get; set; }

public string BikeName { get; set; }

public DateTime Timestamp { get; set; }

public uint DistanceMeters { get; set; }

public uint TimeSeconds { get; set; }

public double MeanSpeedKmh { get; set; }

public double MaxSpeedKmh { get; set; }

public byte Cadence { get; set; }

public uint TripSectionDistanceMeters { get; set; }

public uint TripSectionTimeSeconds { get; set; }


public double DistanceKm => DistanceMeters / 1000.0;

public TimeSpan Duration => TimeSpan.FromSeconds(TimeSeconds);

public TimeSpan TripSectionDuration => TimeSpan.FromSeconds(TripSectionTimeSeconds);

}


5. Userdaten

Die Anwendung speichert automatisch folgende Einstellungen:

  • Fensterposition und -größe

  • Status des Programms (Fenstermodus oder minimiert)

  • Zuletzt verwendeter Tacho (Name und Seriennummer)

  • Liste mit verfügbaren Fahrrädern

  • Sprache (Deutsch/Englisch)*

* Hinweis: die BikeDB2025 ist bisher nur in Deutsch verfügbar

public enum BikeComputerModel

{
